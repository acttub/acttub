# Screen Structure — excer

> Stage 1.4 산출물. 필요한 화면 목록과 각 화면의 핵심 요소.

## 화면 목록 (4개)

| # | 라우트 | 이름 | 목적 | 등급 |
| --- | --- | --- | --- | --- |
| 1 | `/` | 홈 (지도 + 발견) | 진입 직후 모든 발견 행위 | MUST |
| 2 | `/rooms/[slug]` | 연습실 상세 | 결정 + 전환 (전화/예약) | MUST |
| 3 | `/not-found` | 404 | 잘못된 slug, 삭제된 데이터 | MUST |
| 4 | `/admin` | (선택) 운영자 관리 | 시드 추가/수정/검수 — MVP 에선 Drizzle Studio 로 대체 가능 | SHOULD-LATER |

> 주: 모든 라우트는 basePath `/excer` 가 앞에 붙음 → 실제는 `acttub.com/excer/`, `acttub.com/excer/rooms/...`.

## 화면 1 — 홈 (`/`)

**목적**: 진입 즉시 "주변 연습실이 있다"를 보여주고 발견을 시작하게 한다.

### 모바일 레이아웃 (375 ~ 768px)
```
┌─────────────────────────────┐
│ 🎭 excer            [필터 ⚙️]│  ← 상단바 (sticky)
├─────────────────────────────┤
│ [🔎 혜화 / 대학로 / 강남…]    │  ← 검색 입력 (sticky)
│ [📍 내 주변]                  │
├─────────────────────────────┤
│                             │
│                             │
│         🗺  지도 영역        │  ← 화면 60%
│      ● ● ●        ●         │     핀 + 클러스터
│         ●  (24)  ●          │
│                             │
│                             │
├─────────────────────────────┤
│ ▲ 리스트 22곳    [거리▼/가격]│  ← 하단 시트 (드래그)
│ ┌─────────────────────────┐ │
│ │ 사진 │ 혜화 ○○연습실  ▶│ │
│ │      │ 15평·거울·방음 강│ │
│ │      │ ₩15,000/h        │ │
│ └─────────────────────────┘ │
│ ┌─────────────────────────┐ │
│ │ 사진 │ 대학로 ○○스튜디오│ │
│ │      │ 10평·거울        │ │
│ │      │ ₩10,000/h        │ │
│ └─────────────────────────┘ │
└─────────────────────────────┘
```

### 데스크톱 레이아웃 (≥1024px)
```
┌──────────────────────────────────────────────────────────────────┐
│ 🎭 excer        [🔎 혜화/대학로…]   [📍 내 주변]      [필터 ⚙️] │
├───────────────────────────────────┬──────────────────────────────┤
│                                   │ ▲ 리스트 22곳   [거리▼/가격] │
│                                   ├──────────────────────────────┤
│                                   │ ┌──────────────────────────┐ │
│                                   │ │ 사진│혜화 ○○연습실   ▶ │ │
│           🗺  지도                │ │     │15평·거울·방음 강    │ │
│       ● ● ●         ●             │ │     │₩15,000/h           │ │
│           ●  (24)  ●              │ └──────────────────────────┘ │
│                                   │ ┌──────────────────────────┐ │
│                                   │ │ 사진│대학로 ○○스튜디오   │ │
│                                   │ │     │…                   │ │
│                                   │ └──────────────────────────┘ │
│                                   │ ⋮                            │
└───────────────────────────────────┴──────────────────────────────┘
       지도 영역 ~65%                      리스트 패널 ~35%
```

### 핵심 요소
- **상단바**: 로고, 필터 토글 (모바일 우상단 아이콘 / 데스크톱 우측)
- **검색 입력**: 동/지하철역/"내 주변" 버튼. 입력 시 지도 중심·줌 자동
- **지도**: 카카오맵, 핀 + 클러스터, 줌·드래그로 영역 변경 → 리스트 자동 필터
- **하단 시트 / 사이드 리스트**: 현재 지도 영역의 연습실 카드
- **카드**: 썸네일, 이름, 핵심 속성 칩(평수·거울·방음), 시간당 가격
- **필터 패널**: 거울 / 방음 / 평수 / 가격대 / (SHOULD) 조명·대본대·마이크
- **정렬 토글** (SHOULD): 거리순 / 가격순
- **빈 상태**: 필터 0건 → "조건을 풀어보세요" + 가장 가까운 완화 제안

### 상호작용
- 핀 클릭 → 상세 페이지 이동
- 카드 클릭 → 상세 페이지 이동
- 카드 호버 (데스크톱) → 지도 핀 강조
- 검색 입력 → 지도 자동 이동, 리스트 갱신
- 필터 변경 → 지도 + 리스트 동시 갱신 (URL 쿼리에 반영)

## 화면 2 — 연습실 상세 (`/rooms/[slug]`)

**목적**: 발견된 연습실에 대해 결정에 필요한 모든 정보 노출 + 전환 액션(전화/외부 예약).

### 모바일 레이아웃
```
┌─────────────────────────────┐
│ ← 뒤로                       │
├─────────────────────────────┤
│ ┌─────────────────────────┐ │
│ │  사진 캐러셀 (스와이프)   │ │  ← 16:9 비율
│ │      ●○○○                │ │
│ └─────────────────────────┘ │
├─────────────────────────────┤
│ 혜화 ○○연습실               │
│ 종로구 동숭동                │
│ 🕒 09:00~24:00              │
│ 📍 (지도 미니 + 길찾기 링크) │
│                             │
│ ₩15,000 / 시간              │  ← 굵게
│ 🟡 마지막 확인 12일 전 [S1] │
├─────────────────────────────┤
│ ▣ 속성                       │
│ ┌──────────┬──────────────┐ │
│ │ 거울     │ ✅ 있음       │ │
│ │ 방음     │ 🟢 강         │ │
│ │ 평수     │ 15평          │ │
│ │ 조명     │ ⚪ 보통       │ │
│ │ 대본대   │ ✅ 있음       │ │
│ │ 마이크   │ ❌ 없음       │ │
│ └──────────┴──────────────┘ │
├─────────────────────────────┤
│ 📞 [전화하기]                │  ← 풀폭 CTA (sticky 하단)
│ 🔗 [예약 페이지로 이동]      │
└─────────────────────────────┘
```

### 핵심 요소
- **뒤로 가기**: 모바일 좌상단 / 데스크톱은 브레드크럼
- **사진 캐러셀**: 최소 1장, 없으면 placeholder
- **이름 + 주소** + 운영시간
- **시간당 가격** (가장 강조)
- **마지막 확인일 배지** (SHOULD): 30일+ 면 노란색 경고
- **속성 표**: 6항목 — 거울/방음/평수/조명/대본대/마이크
- **CTA 영역** (sticky):
  - "전화하기" → `tel:` 링크
  - "예약 페이지로 이동" → 외부 URL (없으면 숨김 — B6)
- **운영시간 외 표시** (B5): 현재 시각이 운영시간 밖이면 보조 텍스트로 알림

### 상호작용
- 사진 스와이프 / 캐러셀 닷
- 미니지도 클릭 → 카카오맵 길찾기로 새 탭
- CTA 탭/클릭 → 외부 전환 (이탈 → GA 이벤트 기록)

## 화면 3 — 404 (`/not-found`)

**목적**: 잘못된 slug, 삭제된 데이터에 대한 안전한 폴백.

### 레이아웃
```
┌─────────────────────────────┐
│                             │
│           🎭                 │
│                             │
│   요청하신 연습실을 찾을 수    │
│   없어요.                    │
│                             │
│   [지도로 돌아가기 →]         │
│                             │
└─────────────────────────────┘
```

- 단순. acttub 전체 디자인 톤 유지.
- "지도로 돌아가기" 만 단일 CTA.

## 화면 4 — 운영자 관리 (`/admin`) — 선택

**목적**: 시드 POI 추가/수정/마지막 확인일 갱신.

### MVP 결정
- **MVP 1차에는 풀 어드민 UI 생략.**
- 대신: **Drizzle Studio** + Postgres 직접 입력으로 50개 시드 채움.
- 사용자 트래픽 100 DAU 이상 또는 POI 100개 이상이 되면 그때 어드민 UI 트랙.

### 만든다면 (참고용 골격)
```
┌──────────────────────────────────────────┐
│ excer 관리 │ [목록] [추가] [내 계정]      │
├──────────────────────────────────────────┤
│ 검색: [______________]   필터: [전체 ▼]  │
├──────────────────────────────────────────┤
│ ┌────────────────────────────────────┐  │
│ │ 혜화 ○○연습실 │ 12일 전 │ [수정]    │  │
│ │ 대학로 ○○스튜디오 │ 3일 전 │ [수정]  │  │
│ │ … (페이지네이션)                    │  │
│ └────────────────────────────────────┘  │
└──────────────────────────────────────────┘
```

- Clerk role = `admin` 만 접근.
- 수정 폼은 정규화 속성 + 가격 + 사진 URL + "마지막 확인일 갱신" 버튼.

## 공통 UI 규칙
- **디자인 토큰**: Pretendard, neutral surfaces, coral primary (acttub 통일)
- **shadcn/ui** 컴포넌트 우선 사용
- **모바일 우선** + breakpoint 1024 에서 사이드 패널 전환
- **다크 모드** — 시스템 prefers-color-scheme 따름 (MVP 기본)
- **로딩 상태** — skeleton 표시, 지도는 placeholder 그리드
- **에러 상태** — 토스트 + "다시 시도" 액션
