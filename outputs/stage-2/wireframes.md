# Wireframes — excer

> Stage 2.3 산출물. 화면별 모바일/데스크톱 low-fi 와이어프레임 + 핵심 상태.
>
> 범례: `[ ]` 버튼·인풋, `█` 강조 영역, `…` 생략, `▶` 진입 표시, `▲▼` 토글/드래그

## 공통 컴포넌트 — 와이어 레벨

### A. RoomCard (리스트의 카드)
```
┌─────────────────────────────────────┐
│  ┌────┐                             │
│  │ 🖼 │  혜화 ○○연습실           ▶ │
│  │64x64│ ▢15평  🪞거울  🔊방음 강   │  ← Badge 칩

└─────────────────────────────────────┘
```
- 라디우스: `rounded-lg` (14px)
- 호버: `shadow-md` + `-translate-y-0.5`
- 활성(지도 핀 ↔ 카드 호버 연동): `ring-2 ring-ring`

### B. SearchBar
```
모바일:
┌──────────────────────────────────┐
│ 🔎 [동·역 검색          ] [📍]   │  좌: 검색, 우: 내 주변 IconButton

[🔎 ────────── 혜화·대학로·강남… ──────────] [📍 내 주변]
```

### C. RoomPin (지도 마커)
```
   ▼          ▼ (활성)
  ●●●        ███   ← scale-110 + ring
  ●₩●        █15█   ← 가격 또는 평수 라벨 (선택)
  ●●●        ███
```
- 기본: Coral primary 18px 원형
- 활성: 24px + accent ring
- 클러스터: 흰 배경 + 코랄 테두리 + 숫자

### D. AttributeTable (상세)
```
┌─────────┬──────────────┐
│ 거울    │ ✅ 있음        │
├─────────┼──────────────┤
│ 방음    │ 🟢 강          │
└─────────┴──────────────┘
```
